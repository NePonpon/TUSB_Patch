Minecraft ver.1.10.2
The Unusual SkyBlock v12.0.9でお使いできます。
それぞれのディメンションフォルダーの中から選んでください。

　ストラクチャーのNBTファイルの入れ方

１村人を出すワールドのフォルダーを開く
２ structuresフォルダーを開く
３ディメンションフォルダーの中の必要なファイルをstructuresフォルダーに入れる
４準備完了


　村人の出しかた

１ストラクチャーブロックを下のコマンドで入手する

/give @p minecraft:structure_block

２ストラクチャーブロックの設定をする
　１左下の [D] を2回クリックして [L] ロードモードにする
　２右側にある エンティティを含める をオンにする
３ストラクチャー名の所にそれぞれのストラクチャー名(各ディメンションフォルダー内テキストファイル参照)を入れる
４読み込みをクリックする
５出現
